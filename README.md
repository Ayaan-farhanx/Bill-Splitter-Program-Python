# Bill-Splitter-Program-Python
Program (In Python) that calculates a restaurant bill, applies tip percentage and splits the total among multiple people.

# Features
- Calculates total bill amount
- Adding a 25% tip
- Splits the bill between multiple people
- Rounds the final amount per person

# What I practiced
- Variables and arithmetic operations
- Using + - for accumulation
- Working with decimals and rounding
- Formatting printed output
